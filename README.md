# TempleCV

Local-first desktop TempleCV built with Electron, Vite, React, TypeScript, and a ZIP-based `.resume` package format.

## Current Scope

- Structured resume document model
- Versioned schema validation and migration hooks
- Managed package assets for logos, headshots, and screenshots
- HTML/CSS template preview
- Export seams for PDF, ATS PDF, HTML, DOCX, and JSON

## Commands

```powershell
npm install --cache .npm-cache
npm run typecheck
npm start
```
